#System overview 
The smart Care System is an application built for medical clinics that are struggling with synchronizing patient records and scheduling appointments while
ensuring efficiency. The smart care System is designed to specifically replace faulty unreliable systems while ensuring quality patient care. 
This is important because it will empower the health sector to operate smoothly by relieving it of its administrative burdens allowing medical workers to focus
on what matters the most which is the patients. ​


 For our project to work the user  has to install npm packages  and run both the front end and back end using npm run dev 
 In order to have acces  to our database please use prisma commands such npx prisma generate then npx prisma migrate dev and npx prisma studio 
 please go ahead to create an account when logging in as you will be a new user to the system
